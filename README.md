# Sales Insights Data Analysis Project

This project is a comprehensive data analysis and visualization of sales data using Power BI. It aims to provide valuable insights into sales trends, customer behavior, and product performance across various markets. The project leverages data extracted from an SQL database, cleaned and transformed in Power BI, and visualized through interactive dashboards.

---

## Project Overview

### Key Features:

#### Interactive Dashboards:
- **Total Revenue and Sales Quantity overview.**
- **Revenue trends over time (2017-2020).**
- **Top-performing customers, products, and markets.**

#### Data Cleaning:
- Handled missing or inconsistent data directly in Power BI.
- Streamlined the data structure for effective analysis.

#### Detailed Analysis:
- Identified top customers and products driving revenue.
- Analyzed sales distribution across different markets.

---

## Files Included

1. **Charts.pbix**  
   The main Power BI file containing the cleaned data, measures, and interactive dashboards.

## How to Use

### Clone or Download the Repository:
```bash
git clone https://github.com/asomaarooufiniyaa/sales-insights-data-analysis-project.git
```

### Open the Power BI Dashboard:
1. Ensure **Power BI Desktop** is installed on your system.
2. Open the `Charts.pbix` file to explore the dashboard.

### Review the Data:
- **Raw data:** Navigate to `Data/raw_data.csv`.
- **Cleaned data (optional):** Check `Data/cleaned_data.xlsx`.
